# XMFund
Example Fund as a Smart contract. 

```
smart contract XMFund implemented in Solidity.

Contains the following operations:

 purchase()
 sell(uint numberOfUnits)
 close()
 queryValue()
 queryUnits()
 getUnitsAvailable()
 queryUnitPrice()
 setUnitPrice()
 destroy()

```
